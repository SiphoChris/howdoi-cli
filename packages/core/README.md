# @howdoi-cli/core

The engine, renderer, and `howdoi` binary.

Install this first, then install whichever knowledge bases you need.

## Install

```bash
npm install -g @howdoi-cli/core
```

Then add knowledge bases:

```bash
npm install -g @howdoi-cli/unix
npm install -g @howdoi-cli/git
npm install -g @howdoi-cli/ssh
npm install -g @howdoi-cli/docker
npm install -g @howdoi-cli/networking
```

Or install everything at once:

```bash
npm install -g @howdoi-cli/all
```

## Usage

```bash
howdoi <what you want to do>
howdoi --list
howdoi --version
howdoi --help
```

See the [main README](https://github.com/SiphoChris/howdoi-cli) for full documentation.
